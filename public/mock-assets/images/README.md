# Mock Images

Add sample images here for testing. Suggested images:

1. **placeholder-1.jpg** - A simple colored square (e.g., red)
2. **placeholder-2.jpg** - A simple colored square (e.g., blue)
3. **placeholder-3.png** - A simple colored square (e.g., green)
4. **sample-photo.jpg** - Any sample photo

You can use:
- Simple colored images created in any image editor
- Placeholder images from https://via.placeholder.com/
- Sample images from https://unsplash.com/
- Or generate them using Canvas API in code

## Generating Placeholder Images with Code

You can use the Canvas API to generate simple colored placeholders programmatically in the MockProvider.
