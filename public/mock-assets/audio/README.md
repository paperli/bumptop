# Mock Audio Files

Add sample audio files here for testing. Suggested audio:

1. **sample.mp3** - A short audio clip
2. **test.wav** - WAV format (uncompressed)
3. **music.m4a** - M4A format

## Where to Get Sample Audio

- **Free Music Archive**: https://freemusicarchive.org/
- **Freesound**: https://freesound.org/ (Creative Commons sounds)
- **CC0 Sounds**: https://cc0sounds.com/
- Generate a simple sine wave tone using audio software

## Recommended Specs for Testing
- Duration: 10-30 seconds
- Format: MP3 (most compatible) or WAV
- Sample rate: 44.1kHz or 48kHz
- Bitrate: 128kbps or higher for MP3
- File size: < 1MB for fast loading
